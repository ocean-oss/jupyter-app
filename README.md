# Jupyter

This app runs a single Jupyter instance with many pre-installed data science packages.

## Included Packages

This app includes many Python packages for scientific computing:

- dask
- pandas
- numexpr
- matplotlib
- scipy
- seaborn
- scikit-learn
- scikit-image
- sympy
- cython
- patsy
- statsmodel
- cloudpickle
- dill
- numba
- bokeh
- sqlalchemy
- hdf5
- vincent
- beautifulsoup
- protobuf
- xlrd
- bottleneck
- pytables
- ipywidgets
- ipymp
